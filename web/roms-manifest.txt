# clawdputer / retro app — ROM manifest
#
# Hosted on GitHub Pages at:
#   https://chaerem.github.io/clawdputer/roms-manifest.txt
#
# This file is the declarative wishlist of ROMs. Run tools/sync-roms.sh
# on the host to materialize the binaries onto a target dir (typically
# a mounted SD card). The binaries themselves are NEVER committed to
# this repo — only the URLs are.
#
# Format: one entry per line, "filename.gb | URL". '#' for comments.
# The filename on the left is what lands on the SD card (must end in a
# supported extension: .gb, .gbc, plus future cores).
#
# For ROMs you can't legally redistribute (commercial ROMs), use a
# private manifest at web/roms-manifest.private.txt (gitignored) — same
# format, append-loaded by the sync script.
